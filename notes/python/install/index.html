# 开始

macOS系统中自带了python2.7，由于2.7版本较老，并且在2020之后停止维护。因此有必要升级为Python3，但是macOS系统中部分软件是依赖于自带Python2.7的，所以不建议删除之前的老版本，而是让老版本和新版本共存。

通过输入 python指令查看当前版本：

$ python

WARNING: Python 2.7 is not recommended. 
This version is included in macOS for compatibility with legacy software. 
Future versions of macOS will not include Python 2.7. 
Instead, it is recommended that you transition to using 'python3' from within Terminal.

输入python3查看是否安装Python3

$ python3 
Python 3.10.4 (v3.10.4:9d38120e33, Mar 23 2022, 17:29:05) [Clang 13.0.0 (clang-1300.0.29.30)] on darwin

安装完python3后 会发现在命令行中输入python后 显示的仍然是以前的2.7版本，输入python3后显示的才是3.10版本，如果希望做到输入python命令后 显示当前最新版本3.10版本，需要在 ~/.bash_profile文件中添加python 依赖

通过which python3指令查看安装路径

$ which python3
/Library/Frameworks/Python.framework/Versions/3.10/bin/python3

打开文件
vi ~/.bash_profile 

在.bash_profile中添加以来
alias python = "/Library/Frameworks/Python.framework/Versions/3.10/bin/python3" 

读取并执行 
source .bash_profile

届时，再次输入python指令，变换进入python3 环境
